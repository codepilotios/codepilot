## Before any approved post

1. Re-read the live community rules and pinned posts on the day of posting.
2. Confirm that the `codepilotios` account meets community-age, karma, and
   participation requirements without manufacturing engagement.
3. Replace bracketed placeholders with public, tested URLs only.
4. Confirm that the current TestFlight build and public documentation match every
   product claim.
5. Remove any screenshot, log, hostname, token, account name, or local path that
   has not passed the privacy audit.
6. Obtain explicit maintainer approval for the final post and destination.
7. Post once. Do not coordinate votes, repeat the same promotion across threads,
   or use automated posting.
