# CodePilot Release Readiness

Check whether CodePilot is ready for OTA/TestFlight/App Store release work.

Follow `docs/superpowers/plans/2026-07-01-codepilot-launch-agent-system.md`.

Rules:
- Run relevant tests and builds when safe.
- Prepare release notes and blocker summaries.
- Do not submit App Store review or alter pricing/legal metadata without the maintainer.
- If intervention is needed, write a concise escalation note to `ops/agents/escalations/release-readiness.md`.

Report:
- Build/test status.
- Release blockers.
- Metadata or screenshot gaps.
