# CodePilot Setup Audit

Audit CodePilot first-run setup for the Mac app, iOS app, gateway, Cloudflare, permissions, account setup, and docs.

Follow `docs/superpowers/plans/2026-07-01-codepilot-launch-agent-system.md`.

Rules:
- Draft local issue files for setup friction.
- Fix low-risk copy or validation problems in a local branch and commit.
- Do not change auth secrets or create external accounts.
- If intervention is needed, write a concise escalation note to `ops/agents/escalations/setup-audit.md`.

Report:
- Setup paths checked.
- Friction found.
- Local fixes or issue drafts created.
