# CodePilot Health Watch

Check CodePilot gateway health, Cloudflare tunnel reachability, OTA install status, and release readiness.

Follow `docs/superpowers/plans/2026-07-01-codepilot-launch-agent-system.md`.

Rules:
- Do not post publicly.
- Do not change credentials.
- Do not submit App Store builds.
- Create local issue or patch drafts for actionable findings when safe.
- If intervention is needed, write a concise escalation note to `ops/agents/escalations/health-watch.md`.

Report:
- Checks performed.
- Findings.
- Local drafts or commits created.
- Escalations written.
